# Changelog

## 0.1.0 - Initial Public Release

- Initial Rust TUI for everyday Git work.
- Changes, Stash, Branches, Log, Remotes, and Repos panels.
- Diff inspector with hunk navigation, context controls, whitespace modes, and hunk staging support.
- Commit modal with staged-file review, summary warning, and external editor handoff.
- Remote fetch, fast-forward update, push, and credential retry handoff.
- Worktree, submodule, rebase, and conflict helper workflows.
- Background refresh with filesystem watcher and periodic fallback.
- GitHub issue templates, PR template, CI workflow, and publishing checklist.
